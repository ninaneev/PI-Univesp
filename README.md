# PI-Univesp — SIA Intake Portal

Portal web para ingestão segura de arquivos com autenticação, upload, rastreabilidade e trilha de auditoria.

## Objetivo do projeto
Este projeto implementa um portal web onde usuários autenticados podem enviar arquivos manualmente para análise no fluxo SIA (Strategic Intelligence Audit), com persistência de metadados, download controlado e painel administrativo.

## Stack
- Frontend: React + Vite + Flowbite
- Backend: FastAPI
- Banco de dados: PostgreSQL
- Storage: MinIO
- Infra: Docker Compose
- Versionamento: GitHub

## Como o projeto será organizado
- `frontend/` → interface web
- `backend/` → API FastAPI
- `docs/` → documentação, roadmap e relatório
- `.github/` → templates de issues e pull requests

## Fluxo de trabalho
1. Cada tarefa será uma Issue
2. Cada integrante trabalhará em uma branch própria
3. Toda alteração será enviada por Pull Request
4. O progresso será acompanhado via GitHub Projects
